AWS Basics

Simple introduction to Amazon Web Services

What is AWS?

Amazon Web Services (AWS) is a cloud computing platform that provides infrastructure services such as compute, storage, networking, and databases through the internet.

Core AWS Services

EC2

Virtual servers for running applications.

S3

Scalable object storage for files and backups.

RDS

Managed relational databases.

Lambda

Run code without managing servers.
